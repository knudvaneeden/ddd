windowskeyall

1. -To install

     1. -Take the file windowskeyallknud.zip

     2. -And unzip it in any arbitrary directory

     3. -Then run the file zipinstallwindowskeyall.bat

     4. -That will create a new file windowskeyall_knud.zip
         in that directory

     5. -Unzip that latest file in a new arbitrary directory

     6. -Then run the file

          windowskeyall.mac

2. -The .ini file is the local file 'windowskeyall.ini'
    (thus not using the global file 'tse.ini')
