     3. -Then run the file zipinstallprettse.bat

     4. -That will create a new file prettse_knud.zip
         in that directory

     5. -Unzip that latest file in a new arbitrary directory

     6. -Then run the file

          prettse.mac

    (thus not using tse.ini)
