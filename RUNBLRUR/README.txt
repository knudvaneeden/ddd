1. -To install

     1. -Take the file runblrur_knud.zip

     2. -And unzip it in any arbitrary directory

     3. -Then run the file zipinstallrunblrur.bat

     4. -That will create a new file runblrur_knud.zip
         in that directory

     5. -Unzip that latest file in a new arbitrary directory

     6. -Then run the file

          runblrur.mac

    (thus not using tse.ini)
